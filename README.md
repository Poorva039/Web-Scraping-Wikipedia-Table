# Web Scraping Wikipedia Table

This project demonstrates how to scrape a table from a real website (Wikipedia) using Python. The scraped data is then processed and analyzed using pandas. 

## Features
- Extracts a table from Wikipedia using `requests` and `BeautifulSoup`.
- Parses the data into a structured format.
- Uses `pandas` for cleaning and analyzing the data.

## Technologies Used
- Python
- BeautifulSoup
- requests
- pandas

## Install the required libraries:
- pip install -r requirements.txt
